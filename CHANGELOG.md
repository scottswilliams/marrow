## [0.1.0] - 2026-06-14

Marrow v0.1.0 was tagged as the source release on 2026-06-14.

### Distribution

- Source install is the v0.1.0 release channel; see
  [Installing Marrow](docs/install.md).
- Marrow v0.1 targets Unix platforms: Linux and macOS; see the
  [Stability Contract](docs/stability.md).
- Prebuilt binaries and crates.io publication are post-v0.1 follow-up
  channels, not v0.1 release channels.
