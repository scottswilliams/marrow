# Archived Research Reports

This directory contains historical research and audit reports. These files are
not ADRs, lane prompts, current status trackers, or product authority.

Use the accepted ADR packet, the language reference, the CLI/data/serve/backend
and evolution docs, and `docs/roadmap/prototype-to-v1-execution-plan.md` for
canonical truth. When a research report mentions a rejected surface, stale
conflict, or old lane state, treat that mention as archived evidence unless the
canonical docs still state the same rule.
