# Tools

The beta line ships a thin command-line tool while the prototype command
families are refounded lane by lane. Tools consume parser and syntax facts;
they do not define a second language or saved-data model.

- [CLI Reference](cli.md) covers `fmt`, `--version`, `--help`, and the typed
  not-yet-supported response every refounding command name reports.

The prototype project/check/run/test/data/doctor/evolve/backup/restore tooling
and the surface/client/serve commands were deleted at B00; each returns through
its refounding lane. See [Project status](../status.md).
